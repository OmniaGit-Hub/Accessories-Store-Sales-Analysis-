# Accessories-Store-Sales-Analysis-
Delivered a real-world data analysis project for an accessories store client, cleaned and transformed raw sales data using Power Query, built an interactive Power BI dashboard to track key KPIs (Revenue, Orders, Average Revenue), analyzed product performance, customer behavior, and payment methods.
# Tools Used
Power BI

Power Query

Data Modeling

DAX

Data Visualization
# Dashboard Features
Interactive KPI Cards

Dynamic Slicers (Month, City, Payment Method)

Revenue and Sales Trend Visualizations

Product and Customer Insights
# Business Impact
Enabled the client to monitor sales performance through an interactive dashboard.

Identified high-performing products and revenue drivers.

Revealed customer purchasing and payment behavior patterns.

Supported inventory, sales, and marketing decision-making.
